# LumeCard V0.3.0

发布日期：2026-06-18

## ✨ 新增功能

- **数据与配置完全分离** — 数据（KnowledgeBase/Deck/Card/LearningPlan/ReviewLog）与配置（Settings/WebDAV/UI）独立同步，互不干扰
- **独立同步按钮** — 支持「同步全部」「仅同步数据」「仅同步配置」三种模式
- **数据/配置独立文件** — `data.json` 存储学习数据，`config.json` 存储配置
- **旧格式自动迁移** — 检测到旧版同步文件时自动转换为新格式

## 🐛 问题修复

- 修复自动更新功能「立即更新」无响应问题
- 修复 Android 文件选择器无法正常工作
- 修复更新检查文案「发现新版本」→「检查更新」
- 修复更新状态逻辑：初始显示当前版本，点击后才显示检查中
- 修复 WebDAV 同步中 KnowledgeBase 数据丢失问题
- 修复导入导出使用旧版 API 的编译错误

## 🚀 优化改进

- 完善更新检查对话框：支持下载进度、安装提示、错误状态
- 完善导入导出功能：使用文件选择器，支持格式校验
- 优化设置页面 UI：更新按钮文案统一，弹窗风格一致
- 新增 5 种语言更新相关字符串

## ⚠️ 注意事项

- 数据结构升级：ExportManager 新增 DataExport/ConfigExport 数据类
- 同步路径变更：WebDAV 存储从 `lumecard_backup.json` 改为 `data.json` + `config.json`
- 旧版同步文件可自动迁移，无需用户手动操作
